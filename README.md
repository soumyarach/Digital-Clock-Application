# Digital-Clock-Application
# Overview:-
The Digital Clock Application is a simple JavaScript-based project that displays the current time in hours, minutes, and seconds. The clock updates every second in real-time using JavaScript’s setInterval() function. It runs entirely in the browser and requires no backend.

# Features:-
-> Displays real-time digital clock

-> Shows Hours : Minutes : Seconds

-> Automatically updates every second

-> Fully responsive and lightweight

-> Built using HTML, CSS, and JavaScript

# Technologies Used:-
  * HTML — Structure
  * CSS — Styling
  * JavaScript — Time functionality and updates

# How It Works:-
-> JavaScript fetches the current time using the Date() object.

-> Time values are formatted to always show two digits.

-> The UI is updated every second using setInterval().

# How to Run:-
-> Download all files (index.html, style.css, script.js).

-> Open index.html in any browser.

-> The clock starts running automatically.
